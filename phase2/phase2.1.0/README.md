
# Kotonari Phase2 - ユーザー内省支援アシスタント（バックエンド機構）

## 🧭 開発背景と目的

「Kotonari」は、ユーザーがPC上で自然に独り言をつぶやける環境を整え、それを記録・理解・通知してくれる**パーソナル内省支援アプリケーション**です。

2025年6月、プロジェクトは以下のように方針転換されました。

### 🔄 プロジェクト構成の再編

- **パーツ1**（ユーザー側UI）:
  - マイクから音声を取得
  - ローカルに保存
  - 必要に応じて相槌を返す機能を持つ

- **パーツ2**（このフォルダ）:
  - パーツ1から渡された録音やテキストを処理
  - アラート条件を満たした場合、LINEやSlackに通知
  - 将来的には感情理解やリマインドなど「気づき」を与える機構へ拡張

---

## 🔨 現在の実装（Phase 2.1.0）

### ✅ 機能概要

- `.txt` ファイルをドラッグ＆ドロップ
- 中身のテキストを表示
- 名詞・動詞・形容詞などの品詞を色分けして表示（形態素解析）

### 📦 使用技術

- [Streamlit](https://streamlit.io/) - フロントエンド
- [Janome](https://github.com/mocobeta/janome) - 形態素解析器（純Python）

---

## 🚀 起動方法

pip install -r requirements.txt

## 👨‍💻 担当範囲について

私は「パーツ2（バックエンド／処理系）」を担当します。
